# OrbitLab

Protótipo estático do OrbitLab, uma plataforma para reservar capacidade experimental em laboratórios espaciais sob demanda.

## Como abrir

Abra `index.html` no navegador ou rode:

```bash
npm run dev
```

Depois acesse `http://127.0.0.1:4173/`. O app não precisa instalar dependências.

## Telas implementadas

- Login e cadastro institucional
- Tela inicial com recomendações
- Catálogo de plataformas orbitais
- Detalhes da plataforma
- Cadastro de experimento
- Compatibilidade e escolha de plataforma
- Reserva da janela experimental
- Dashboard com estatísticas e histórico
- Cadastro de capacidade experimental para operadoras
